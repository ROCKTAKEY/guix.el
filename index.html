About

Emacs-Guix (aka guix.el) is the Emacs interface for GNU Guix package manager.

Emacs-Guix provides various features, in particular it allows you to manage your Guix profile(s): to install, upgrade and remove packages, to switch and remove profile generations, to display all available info about packages and to do many other things.

You can install it from MELPA or using Guix:

guix package -i emacs-guix

Source code of Emacs-Guix: https://github.com/alezost/guix.el

Source code of this site: https://github.com/alezost/guix.el/tree/gh-pages

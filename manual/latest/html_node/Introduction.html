1 Introduction

Emacs-Guix (also known as “guix.el”) provides various interfaces and tools related to the GNU Guix package manager.

Call M-x guix-about to see a splash screen similar to M-x about-emacs.

Call M-x guix-help to get a summary of all the available commands (see Help).

In short, Emacs-Guix provides the following features:

  • Interfaces for:
    • - profiles (see Profiles)
    • - profile generations (see Generations)
    • - packages (see Packages)
    • - package licenses (see Package Licenses)
    • - package locations (see Package Locations)
    • - Hydra build farm (see Hydra)
  • Magit-like popup interface for all guix shell commands (see Popup Interface).
  • Modes to view logs of package builds (see Build Log Mode).
  • Minor mode to “prettify” store file names (see Prettify Mode).
  • Shell completions for all guix commands and options (see Completions).
  • Minor mode with additional functionality for scheme-mode to work with Guix Scheme files (see Development).

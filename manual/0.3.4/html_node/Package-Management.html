3 Package Management

Emacs-Guix provides a visual interface for routine package management tasks, pretty much like the guix package command (see Invoking guix package in The GNU Guix Reference Manual). Specifically, it makes it easy to:

  • browse and display packages and generations;
  • search, install, upgrade and remove packages;
  • display packages from previous generations;
  • do some other useful things.
• Packages:  Interface for packages.
• Profiles:  Interface for profiles.
• Generations:  Interface for profile generations.

3.1 Packages

There are multiple commands to display a list of Guix packages. You can either perform some actions in such “package-list” buffer, or “describe” package(s) with RET and do what you need in “info” buffer.

• Package Commands:  M-x guix-… to show packages.
• Package Keys:  Key bindings in “package-list” buffer.
• More Package Commands:  Other M-x guix-… related to packages.
